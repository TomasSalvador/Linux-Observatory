I created Linux Observatory to gain practical experience with the areas most commonly found in Linux systems and kernel engineering roles. Rather than 
focusing on a single feature, the project was designed as a learning platform covering Linux internals, concurrency, networking, debugging, automation,
packaging, and kernel development.

The goal is not to replace existing monitoring tools, but to understand how such tools are built and how software interacts with the operating system
at different layers—from kernel modules and /proc interfaces to user-space services and command-line applications.
